On with the questions....

1. Selektor adalah katakunci dan simbol yang digunakan pada CSS untuk menyeleksi atau memilih elemen HTML.

2. Seperti fungsi RGB, fungsi HSL juga memiliki nilai alpha untuk menyatakan nilai transparansi warna. Nilai alpha dapat diberikan dari 0.0~1.0. Semakin mendekati angka satu, warnanya akan semakin menghilang dan semakin mendekati angka nol, warnanya akan semakin terang.

3. Selektor class adalah selektor yang memilih elemen berdasarkan nama class yang diberikan. Selektor class dibuat dengan tanda titik di depannya.

4. Fungsi RGB memiliki tiga parameter, yaitu nilai untuk merah (red), hijau (green), dan biru (blue). Nilai yang diberikan pada parameter tersebut dimulai dari angka 0~255. Semakin besar nilai yang diberikan, semakin lemah nilai warna tersebut.

5. Selektor ID hampir sama dengan class. Bedanya, ID bersifat unik. Hanya boleh digunakan oleh satu elemen saja. Selektor ID ditandai dengan tanda pagar (#) di depannya.
